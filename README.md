# Praveen Kumar - AI Agentic Engineer Portfolio

Welcome to my personal portfolio! This is an interactive, stunning web experience built to showcase my expertise in AI/ML, Data Science, Cloud Infrastructure, and Agentic Engineering. 

The site utilizes modern web technologies, fluid animations, and interactive 3D elements powered by WebGL/three.js to create a highly engaging visual narrative.

## ✨ Features

- **Immersive 3D Experience:** Custom integration of `@react-three/fiber` and `@react-three/rapier` featuring a physics-driven tech stack showcase of modern AI, Database, and Cloud computing logos.
- **Fluid GSAP Animations:** Complex scrolling sequences, text reveals, and timeline effects powered by `gsap` and `gsap/ScrollTrigger`.
- **Dynamic Projects Carousel:** An interactive work showcase scaling high-resolution application screenshots seamlessly on both desktop and mobile views.
- **Agentic AI Profile:** Custom career timeline, skill categories, and custom graphics tailored perfectly to the AI and Systems Engineering industry.

## 📦 Getting Started

### Prerequisites
Make sure you have [Node.js](https://nodejs.org/) installed on your machine.

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Praveenkumarchilukuri/Portfolio.git
   cd Portfolio
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   ```

4. **View the site:**
   Open your browser and navigate to `http://localhost:5173` (or whatever URL Vite provides in your terminal) to check out the real-time compilation!
